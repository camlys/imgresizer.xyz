# **App Name**: ImgResizer

## Core Features:

- Image Upload & Preview: Allow users to effortlessly upload image files via drag-and-drop or file selection, immediately displaying a clear preview of the original image.
- Flexible Resizing Options: Enable users to specify desired image dimensions by inputting exact width and height values, adjusting by percentage, or selecting from a set of common preset sizes.
- Aspect Ratio Control: Provide a toggle to easily lock or unlock the aspect ratio, allowing users to either maintain proportionality or independently adjust image width and height.
- Smart Compression Recommendations: An AI tool that analyzes the uploaded image's content to suggest optimal compression settings, balancing file size reduction with visual quality preservation.
- Instant Resized Preview: Display a real-time, interactive preview of the image after applying resizing and compression settings, allowing users to verify results before final download.
- Download Resized Image: Offer a straightforward option to download the final resized and optimized image in common formats such as JPG or PNG.
- Fully Responsive Design: Ensure the application's user interface and functionality adapt seamlessly and provide an optimal experience across all device sizes, including desktops, tablets, and mobile phones.

## Style Guidelines:

- Primary color: A sophisticated, medium-dark blue-grey (#626B78) for a professional and modern foundation. Chosen to evoke a sense of digital precision and clarity.
- Background color: A nearly-white, highly desaturated off-white (#F9FAFB) with a hint of blue-grey, ensuring a clean and lightweight aesthetic.
- Accent color: A clean, vibrant cyan (#52C7DB) used for interactive elements and key highlights, symbolizing efficiency and a fresh user experience. It provides strong contrast to the primary color.
- Font for headlines and body text: 'Inter', a grotesque sans-serif. Its modern, objective, and highly legible characteristics make it ideal for a functional, lightweight tool, ensuring readability across all screen sizes.
- Employ a minimalist, line-art style for icons to maintain a clean aesthetic and provide clear, intuitive visual cues for all actions, enhancing usability without clutter.
- A spacious and intuitive layout, prioritizing easy access to core resizing functions. Adheres to mobile-first principles with clear content hierarchy to optimize user experience and facilitate discoverability across devices, while also supporting SEO through a clean structure.
- Subtle and functional animations to provide visual feedback for user interactions, such as highlighting drag-and-drop zones, indicating upload progress, or confirming download completion, maintaining a fast and responsive feel.
